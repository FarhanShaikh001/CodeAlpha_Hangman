# 🎮 Hangman Game - Python

## 📌 About Project

This is a simple **console-based Hangman Game** made using Python.

The computer randomly selects a word, and the player has to guess the word by entering one letter at a time. The game shows correct and wrong guesses until the player wins.

This project was created as part of the **CodeAlpha Python Internship Task**.

---

## ✨ Features

- Random word selection
- User letter input
- Hidden word display using `_`
- Correct and wrong guess checking
- Win condition detection
- Simple beginner-friendly gameplay

---

## 🛠️ Technologies Used

- Python
- Random Module
- Loops
- Conditions
- Lists
- Strings
